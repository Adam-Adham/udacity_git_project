Bikeshare Project



Project Overview



This project analyzes bikeshare data from Chicago, New York City, and Washington. The program allows the user to choose a city, month, and day, then displays statistics about travel times, stations, trip duration, and users.



Files Used



\- bikeshare.py: Python script used to run the bikeshare analysis.

\- chicago.csv: Dataset for Chicago.

\- new\_york\_city.csv: Dataset for New York City.

\- washington.csv: Dataset for Washington.



The CSV files are used locally but are not uploaded to GitHub because they are included in .gitignore.



How to Run using terminal:

python bikeshare.py



This project was completed as part of the Udacity Programming for Data Science course.
